Dasar Teori

Multimedia pada web meliputi suara, musik, video, film, dan animasi. Multimedia hadir dalam berbagai format, seperti gambar, musik, suara, video, rekaman, film, animasi, dan masih banyak lagi. Halaman web sering kali mengandung elemen-elemen multimedia dari berbagai jenis dan format.

Bentuk-bentuk Video

Format File Ekstensi Deskripsi
MPEG .mpg, .mpeg Dikembangkan oleh Moving Pictures Expert Group. Format video populer pertama di web. Tidak didukung dalam HTML5.
AVI .avi Dikembangkan oleh Microsoft. Digunakan dalam kamera video dan perangkat keras TV. Tidak didukung di browser web.
WMV .wmv Dikembangkan oleh Microsoft. Biasa digunakan dalam perangkat TV. Tidak dimainkan di browser web.
QuickTime .mov Dikembangkan oleh Apple. Digunakan dalam kamera video dan perangkat keras TV. Tidak didukung di browser web.
RealVideo .rm, .ram Dikembangkan oleh Real Media. Masih digunakan untuk video online, tetapi tidak diputar di browser web.
Flash .swf, .flv Dikembangkan oleh Macromedia. Memerlukan plug-in tambahan untuk dimainkan di browser web.
Ogg .ogg Dikembangkan oleh Xiph.Org Foundation. Didukung oleh HTML5.
WebM .webm Dikembangkan oleh Mozilla, Opera, Adobe, dan Google. Didukung oleh HTML5.
MP4 .mp4 Dikembangkan oleh Moving Pictures Expert Group. Berdasarkan QuickTime. Direkomendasikan oleh YouTube. Didukung oleh HTML5.

Format File Ekstensi Deskripsi

MP4 .mp4 Sebenarnya format video, tetapi dapat digunakan untuk audio. Didukung oleh semua browser HTML5.

YouTube Videos

Cara termudah untuk memainkan video di HTML adalah dengan menggunakan YouTube. Anda tidak perlu mengonversi video ke berbagai format agar dapat diputar di semua browser.

Menampilkan Video YouTube di HTML

Untuk memutar video Anda di halaman web, lakukan langkah-langkah berikut:

  1. Unggah video ke YouTube.
  2. Catat ID video (contoh: tgbNymZ7vqY).
  3. Tetapkan elemen <iframe> di halaman web Anda.
  4. Gunakan atribut src untuk menghubungkan ke URL video.

Contoh Menggunakan iFrame

<iframe width="420" height="315" src="https://www.youtube.com/embed/tgbNymZ7vqY"></iframe>

API Geolocation

Geolocation API digunakan untuk mendapatkan lokasi geografis pengguna. Geolocation tersedia di browser modern, memungkinkan pengembang untuk memanfaatkan data lokasi dalam aplikasi web.

Contoh Penggunaan Geolocation API

Contoh penggunaan API Geolocation dalam HTML untuk mendapatkan posisi geografis pengguna:

<button onclick="getLocation()">Coba Dapatkan Lokasi</button>
<p id="demo"></p>

<script>
function getLocation() {
    if (navigator.geolocation) {
        navigator.geolocation.getCurrentPosition(showPosition);
    } else {
        document.getElementById("demo").innerHTML = "Geolocation tidak didukung di browser ini.";
    }
}

function showPosition(position) {
    document.getElementById("demo").innerHTML = 
    "Latitude: " + position.coords.latitude + 
    "<br>Longitude: " + position.coords.longitude;
}
</script>

API Drag and Drop

Drag and Drop API memungkinkan pengguna untuk menyeret elemen-elemen pada halaman web dan menjatuhkannya di lokasi yang diinginkan. Fitur ini sering digunakan dalam aplikasi yang membutuhkan interaktivitas tinggi.

Contoh Penggunaan Drag and Drop

Contoh sederhana Drag and Drop menggunakan elemen HTML:

<div id="div1" ondrop="drop(event)" ondragover="allowDrop(event)" style="width:350px;height:70px;padding:10px;border:1px solid #aaaaaa">
  Drop item here
</div>

<img id="drag1" src="img_logo.png" draggable="true" ondragstart="drag(event)" width="88" height="31">

<script>
function allowDrop(ev) {
  ev.preventDefault();
}

function drag(ev) {
  ev.dataTransfer.setData("text", ev.target.id);
}

function drop(ev) {
  ev.preventDefault();
  var data = ev.dataTransfer.getData("text");
  ev.target.appendChild(document.getElementById(data));
}
</script>
